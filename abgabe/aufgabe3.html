Webseiten

HTML ist die grundlegende Sprache, um Webseiten zu strukturieren. Es beschreibt die Inhalte und ermöglicht die Integration von Text, Bildern und anderen Medien.

CSS kümmert sich um das Design und Layout von Webseiten. Es ermöglicht Entwicklern, das Aussehen von HTML-Elementen zu steuern und ansprechende Designs zu erstellen.

Frameworks wie Bootstrap bieten vordefinierte Layouts und Designs, die die Entwicklung von Webseiten erleichtern und beschleunigen.

Internet

HTTP ist das Protokoll, über das Webbrowser und Server kommunizieren, um Webseiten zu laden und Daten auszutauschen.

TCP sorgt für die zuverlässige Übertragung von Datenpaketen über Netzwerke. Es stellt sicher, dass alle Daten korrekt und vollständig ankommen.

Die Internetschicht kümmert sich um das Routing von Daten über Netzwerke, damit sie ihren Zielort erreichen.
